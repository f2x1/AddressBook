# 地址簿应用

一个简单易用的地址簿应用，支持联系人管理、收藏功能、多种联系方式以及Excel导入导出。

## 功能特性

### 1. 基本功能
- ✅ **联系人管理**：添加、编辑、删除联系人
- ✅ **收藏联系人**：标记常用联系人，方便快速查找
- ✅ **多种联系方式**：支持电话、邮箱、微信、地址等多种联系方式
- ✅ **Excel导入导出**：方便备份和迁移联系人数据
- ✅ **响应式设计**：适配各种屏幕尺寸

### 2. 高级功能
- 📱 **标签页切换**：可查看全部联系人或仅查看收藏联系人
- 💾 **本地存储**：联系人数据存储在浏览器本地，无需服务器
- 🎨 **现代UI设计**：采用现代化的设计风格，支持深色/浅色主题切换

## 使用方法

### 直接打开

1. 下载项目文件到本地
2. 用浏览器打开 `index.html` 文件即可使用

### 本地服务器（推荐）

```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js的http-server
npx http-server -p 8000
```

然后在浏览器访问：`http://localhost:8000`

## 功能说明

### 添加联系人
1. 点击「添加联系人」按钮
2. 填写姓名和至少一种联系方式
3. 点击「保存」按钮

### 添加多种联系方式
1. 在添加/编辑联系人表单中
2. 点击「添加联系方式」按钮
3. 选择联系方式类型并填写内容
4. 如需删除，点击对应联系方式后的「删除」按钮

### 收藏联系人
1. 在联系人卡片中
2. 点击姓名右侧的星星图标
3. 星星变实心表示已收藏，空心表示未收藏

### 查看收藏联系人
1. 点击「收藏联系人」标签页
2. 仅显示已收藏的联系人

### 导出Excel
1. 确保地址簿中有联系人
2. 点击「导出Excel」按钮
3. 浏览器会自动下载Excel文件

### 导入Excel
1. 点击「选择文件」按钮
2. 选择符合格式的Excel文件
3. 联系人会自动添加到地址簿中

## Excel文件格式要求

### 导出格式
- 第一列为「姓名」
- 后续列为各种联系方式（电话、邮箱、微信、地址等）
- 如果同一类型有多个联系方式，会自动添加序号（如：电话、电话2、电话3）

### 导入格式
- 第一行必须包含「姓名」列
- 其他列名可包含联系方式类型（如：电话、邮箱、微信、地址）
- 支持中文列名和英文列名
